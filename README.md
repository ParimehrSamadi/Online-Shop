Laletka E-Store 

A dynamic, fully responsive e-commerce storefront built with vanilla JavaScript. This project integrates a real-world API to fetch products and manages a persistent shopping cart system using browser storage.

Key Features

Dynamic Product Integration
- Real-time Data: Uses the FakeStoreAPI to fetch a wide variety of products, including images, titles, descriptions, and prices.
- Category Filtering: A dynamic dropdown menu that allows users to filter products by category.
- Price Range Filtering: A custom-built price filter to narrow down products based on the user's budget.
- Loading Experience: A custom-designed CSS animation (jumping dots) that appears during data fetching to enhance the User Experience (UX).

Advanced Shopping Cart System

- Persistent Storage: Leverages `localStorage` to keep track of selected items, ensuring that the cart data remains intact even after page refreshes or navigation.
- Dynamic Badge: The cart icon in the navigation bar updates in real-time to show the total number of items selected.
- Cart Management: A dedicated checkout page where users can view their selected items, remove products, and see the calculated total price.

Modern UI/UX Design

- Fully Responsive: Built with a "Mobile-First" approach using CSS Media Queries to ensure a seamless experience on smartphones, tablets, and desktops.
- Interactive Elements: Hover effects on product cards and navigation links for a more professional feel.
- Custom Styling: Clean and modern aesthetics with a focus on readability and intuitive navigation.

Technical Stack

- Frontend: HTML5, CSS3, JavaScript (ES6+)
- Styling: Bootstrap 5, Custom CSS (Keyframes, Media Queries)
- API: Fetch API (Asynchronous data handling)
- Storage: Web Storage API (LocalStorage)
- Icons: Bootstrap Icons

Project Structure

- `index.html` / `buy.html`: The main structure of the home and cart pages.
